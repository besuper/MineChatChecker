# MineChatChecker
Check if player use or not MineChat. You can kick or send a Message to Admin. Don't check with Chat !

Default Permission: 
(You can change permission in ConfigFile)
- minechat.bypass | For bypass the checker
- minechat.alert | For receive alert when a player join with MineChat
- minechat.* | All permission
- minechat.command.help | For view help message in commande
- minechat.command.info | For execute info command
- minechat.command.check | For execute check command

Command:
/minechat info <player> | Get information of a player
/minechat check <player> | Check player

Fonctionnality:
- Alert message
- Configurable message
- Kick
- Configurable permission
- Checking every 1 minute
